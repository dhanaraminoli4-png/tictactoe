# tic_tac_toe

This is a simple command-line Tic-Tac-Toe game written in Python.  
It is a beginner-friendly project created to practice Python basics such as
lists, loops, functions, and condition checking.

## Features
- Two-player game (X and O)
- Players choose positions using numbers 1–9
- Automatically checks for a winner or a tie
- Displays the board after every move

## How the Board Works
The game board is stored as a list with 9 elements:

Indexes:
0 | 1 | 2  
3 | 4 | 5  
6 | 7 | 8  

Each position can contain:
- `"X"` → Player X
- `"O"` → Player O
- `" "` → Empty space
